自动让 Maven 关联 jar 的源码

      其实 eclipse 的 Maven 插件就提供了这个功能，但是可能有许多开发人员没关注这个功能，所以这个分享给那些不太清楚
      这个功能的同学，配置很简单，只需要勾选windows-Preferences-Maven-Download Artifact Sources 这个选项就可以了 。




如果出现不编译 class 或 .properties，那么 修改 java build path -> resource -> include 删除 include
